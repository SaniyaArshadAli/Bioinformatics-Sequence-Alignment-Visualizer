# Bioinformatics Sequence Alignment Visualizer

An interactive educational tool for visualizing sequence alignment algorithms (Needleman-Wunsch and Smith-Waterman) with step-by-step matrix filling and traceback visualization.

## Features

✅ **Interactive Input**: Enter DNA/protein sequences and customize scoring parameters
✅ **Dual Algorithms**: Support for both global (Needleman-Wunsch) and local (Smith-Waterman) alignment
✅ **Step-by-Step Visualization**: Watch the matrix fill cell-by-cell with explanations
✅ **Traceback Path**: Visualize the optimal alignment path through the matrix
✅ **Educational Mode**: Detailed explanations at each step of the algorithm
✅ **Performance Analysis**: Time and space complexity breakdown
✅ **Export Results**: Download alignment results as text files
✅ **Responsive Design**: Works on desktop and mobile devices
