View Results: See the final alignment, score, and traceback path

Export: Download results for documentation

Algorithm Explanation
Needleman-Wunsch (Global Alignment)
Aligns entire sequences from beginning to end

Uses dynamic programming to find optimal alignment

Best for sequences of similar length

Smith-Waterman (Local Alignment)
Finds the best matching subsequence

Allows for local similarities in sequences

Ideal for finding conserved domains

Scoring Matrix
Match Score: Positive reward for matching characters

Mismatch Penalty: Negative score for mismatches

Gap Penalty: Negative score for insertions/deletions

Complexity Analysis
Time Complexity: O(m×n) where m and n are sequence lengths

Space Complexity: O(m×n) for standard implementation

Optimization: Hirschberg's algorithm can reduce space to O(min(m,n))

Educational Value
This tool is designed to help students and researchers understand:

How dynamic programming works for sequence alignment

The difference between global and local alignment

How scoring matrices are filled and traceback works

The computational complexity of alignment algorithms

Interactive Testing Script
You can also create a test script to run multiple cases:

python
# test_aligner.py
import requests
import json

def test_alignment(seq1, seq2, algorithm, match, mismatch, gap):
    """Test alignment with given parameters"""
    url = "http://localhost:5000/visualize"
    data = {
        'seq1': seq1,
        'seq2': seq2,
        'algorithm': algorithm,
        'match_score': match,
        'mismatch_penalty': mismatch,
        'gap_penalty': gap
    }
    
    response = requests.post(url, data=data)
    if response.status_code == 200:
        print(f"✓ Test passed for {algorithm}")
        return True
    else:
        print(f"✗ Test failed: {response.json().get('error', 'Unknown error')}")
        return False

# Run test suite
test_cases = [
    ("GATTACA", "GATTACA", "needleman_wunsch", 2, -1, -1),
    ("GATTACA", "GATACA", "needleman_wunsch", 2, -1, -1),
    ("TACGGGCCCTAT", "GGGCCCTATTTT", "smith_waterman", 2, -1, -2),
    ("A", "A", "needleman_wunsch", 1, -1, -1),
]

for seq1, seq2, algo, match, mismatch, gap in test_cases:
    print(f"\nTesting: {seq1} vs {seq2} ({algo})")
    test_alignment(seq1, seq2, algo, match, mismatch, gap)
